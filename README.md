# FoodInfo

## Архитектура: MVP

## Стек технологий:

Язык программирования – Swift

Пользовательский интерфейс – UIKit

Изображение с камеры устройства – AVFoundation

Локальное хранилище данных – Core Data

База данных - Cloud Firestore
